Search*

Each break will have certain qualities, this is where you can filter and find the perfect wave for you. For example, if you are learning how to surf you might want to you use the filters surf-level: beginners crowds: friendly and or small Where as if you are a advance surfer you might use the filters: break-type: point wave-type: barreling prefered-tide: low

Map*

This will be where you see a visual map of all the avaiable surf breaks and info about them. Such as synposis of the break (type, crowds), surfer-level (beginner, intermediate, advance), best conditions for this break (swell direction, tides, etc.)

Current

This is the #1 feature to find the perfect wave for the today's conditions. It allows you to easily mentally compare different breaks and waves.

Forecast*

Current gives you the best information if you want to go surfing today or tomorrow. Forecast gives you the best information if you want to plan your next surf session. This information is based more on the swell predictions.

Preferences*

Each surfer is unique, and I wanted each to have ability to customize this app to how they want to use. For example, wetsuit whether might be all the time or when water-temp: > 65%F wavesize might be 2ft or 6ft mainland or head-high push notifications when the swell-directions: South or wave-energy: > 50kj
